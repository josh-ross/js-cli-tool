# Usage
<!-- usage -->
```sh-session
$ npm install -g js-cli-tool
$ js-cli-tool COMMAND
running command...
$ js-cli-tool (-v|--version|version)
js-cli-tool/0.0.0 darwin-x64 node-v14.15.4
$ js-cli-tool --help [COMMAND]
USAGE
  $ js-cli-tool COMMAND
...
```
